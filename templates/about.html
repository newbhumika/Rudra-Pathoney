About

This lightweight dashboard visualizes Cowrie honeypot activity. It tails logs from docker logs -f cowrie or local files in ./cowrie/log/ and stores parsed events in SQLite.

  • Run: python3 app.py then open http://localhost:5000.
  • Env vars: INGEST_FROM (auto|docker|files|none), COWRIE_DOCKER_NAME, COWRIE_LOG_DIR.
  • CSV: Export filtered logs via the Logs page.

Built with Flask, Bootstrap, Chart.js, and DataTables.
